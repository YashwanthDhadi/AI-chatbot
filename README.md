# AI Chatbot (Flask + Gemini API)

## Features
- AI chatbot using Gemini API
- Clean UI
- Real-time chat

## Setup

1. Clone the repo
2. Create a `.env` file:
   GEMINI_API_KEY=your_api_key_here

3. Install dependencies:
   pip install -r requirements.txt

4. Run:
   python app.py

## Tech Stack
- Flask
- HTML/CSS/JS
- Gemini API
