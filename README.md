# AI Chatbot (Flask + Gemini API)

An AI-powered chatbot built using Flask and the Gemini API, featuring a clean and responsive user interface for real-time conversations. The project demonstrates API integration, backend development, and interactive web application design using Python and Flask.

## Features

* AI chatbot powered by Gemini API
* Real-time conversational responses
* Clean and user-friendly interface
* Secure API key management using environment variables
* Lightweight and easy to deploy

## Tech Stack

* Python
* Flask
* HTML
* CSS
* JavaScript
* Gemini API

## Setup

1. Clone the repository.
2. Create a `.env` file and add your API key:

   ```env
   GEMINI_API_KEY=your_api_key_here
   ```
3. Install dependencies:

   ```bash
   pip install -r requirements.txt
   ```
4. Run the application:

   ```bash
   python app.py
   ```
